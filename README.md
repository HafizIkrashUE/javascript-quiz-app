# JavaScript Quiz App

A beginner-friendly Quiz Application built using HTML, CSS, and Vanilla JavaScript. It allows users to answer multiple-choice questions and displays the final score at the end.

## Features

- 5 multiple-choice questions
- Instant answer checking
- Highlights correct and incorrect answers
- Displays final score
- Play Again button

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## JavaScript Concepts Used

- Arrays
- Objects
- Functions
- DOM Manipulation
- Event Listeners
- createElement()
- appendChild()
- forEach()
- if...else
- innerHTML
- textContent
